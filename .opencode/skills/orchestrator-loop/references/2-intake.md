## Stage 2: INTAKE

Understand the work item.

| Step | Tool | What it does |
|---|---|---|
| 2a | `gh-issue_view`(number=N, includeComments=true) | Get title, body, comments, labels |
| 2b | `read`(package.json) | Get build/lint/test commands |
| 2c | `todowrite` | Mark INTAKE complete; add 1-3 slice items |

**Keep slices minimal.** If the work touches one area (e.g., a single block), use 1 slice. Only decompose if the work spans 2+ unrelated areas.
